# SCL_INDETERMINATE

Явный выбор неопределённого (неинициализированного) хранилища.

- Заголовок: `#include <scl/utility/attribute/indeterminate.h>`
- Агрегатор: `#include <scl/utility/attribute.h>`

---

## SCL_INDETERMINATE

Помечает объявление переменной как имеющей неопределённое начальное значение,
явно отказываясь от нулевой или дефолтной инициализации, которую компилятор
иначе вставил бы. Это позволяет компилятору полностью исключить инструкцию
инициализации — что важно в плотных циклах или горячих путях, где временная
переменная всегда записывается до первого чтения.

Соответствует атрибуту C++26 `[[indeterminate]]`. На компиляторах без его
поддержки макрос раскрывается в пустую строку — переменная инициализируется
обычным образом, без неопределённого поведения и без выигрыша в
производительности.

Размещается перед типом в объявлении локальной переменной.

> **Безопасность:** Чтение переменной до записи в неё — **неопределённое
> поведение**. Используйте только тогда, когда переменная безусловно
> присваивается до первого чтения на каждом пути выполнения.

### Определение наличия

| Условие | Раскрытие |
|---------|-----------|
| `__has_cpp_attribute(indeterminate)` (C++26) | `[[indeterminate]]` |
| Ни одно не выполнено | *(пусто — обычная инициализация)* |

### Использование

```cpp
#include <scl/utility/attribute/indeterminate.h>

void encode(const float* src, std::uint8_t* dst, std::size_t n) {
    SCL_INDETERMINATE std::uint8_t byte;  // записывается на каждой итерации
    for (std::size_t i = 0; i < n; ++i) {
        byte = static_cast<std::uint8_t>(src[i] * 255.0f);
        dst[i] = byte;
    }
}
```

---

## Примечания

- На компиляторах без поддержки `[[indeterminate]]` запасной вариант — пустая
  строка: код корректно компилируется и работает, просто без подсказки
  оптимизатору.
- В отличие от `SCL_ASSUME`, здесь нет вычисляемого выражения во время
  выполнения — атрибут влияет только на то, генерируется ли код инициализации.
- Макрос можно переопределить до включения заголовка через
  `#define SCL_INDETERMINATE`.
